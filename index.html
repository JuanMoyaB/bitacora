2. Crea proyecto → API y servicios → Credenciales
3. Crea ID de cliente OAuth 2.0 tipo Aplicación web
4. Habilita la Google Drive API
5. En "Orígenes autorizados de JavaScript" agrega el origen donde abres este archivo
6. Pega el Client ID aquí abajo
